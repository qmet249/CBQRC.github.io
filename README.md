# CBQRC

Chungbuk Quantum Research Center official website.

## File Structure

- `home.html`, `members.html`, `history.html`, `news.html`, `media.html`, `contact.html`: Korean pages kept at the repository root for direct GitHub Pages publishing.
- `*-en.html`: English pages.
- `styles/base/site.css`: Minimal shared layout and responsive styles.
- `scripts/core/site.js`: Shared rendering and UI behavior.
- `scripts/data/site-data.js`: Korean content data for news, media, history, members, and page cards.
- `scripts/data/site-data-en.js`: English content data.
- `scripts/i18n/site-ui-en.js`: English UI copy overrides.
- `assets/brand`: CBQRC logo file.
- `assets/people`: Profile images.
